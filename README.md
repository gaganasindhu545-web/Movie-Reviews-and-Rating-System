# 🎬 Movie-Reviews-and-Rating-System 

## Description
This project is built using MERN Stack. It allows users to view movies and give reviews.

## Features
- Login & Registration
- View Movies
- Add Reviews
- Ratings

## How to Run
1. Download the project
2. Open in VS Code
3. Run backend and frontend
